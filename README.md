# 2048
In the most case you should use `vue-loader` and `stylus-loader`
## Installation
```
npm install yqy2048
```

```javascript
import yqy2048 from 'yqy2048'
```

## API

null

## How to play

PC: use the `up down left right` key to move the box.

Mobile: use the finger to move the box.
